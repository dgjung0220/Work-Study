### np.where()

- cond 값에 따라 xarr, yarr 을 선택적으로 사용하고자 할 때.
```python
xarr = np.array([1.1, 1.2, 1.3, 1.4, 1.5])
yarr = np.array([2.1, 2.2, 2.3, 2.4, 2.5])
cond = np.array([True, False, True, True, False])

result = np.where(cond, xarr, yarr)
# array([1.1, 2.2, 1.3, 1.4, 2.5])
```

- 조건문
```python
arr = np.random.randn(4,4)
# array([[-0.33064906,  1.31710691, -1.20929661, -0.46463464],
#         [ 0.35413982,  0.55298365, -2.28132049, -0.07881734],
#         [ 0.93925669, -1.60864988,  1.56677965, -0.38871597],
#         [-0.49153513, -0.36618016,  1.19332117, -0.27166206]])

np.where(arr > 0, 2, -2)
# array([[-2,  2, -2, -2],
#        [ 2,  2, -2, -2],
#        [ 2, -2,  2, -2],
#        [-2, -2,  2, -2]])

np.where(arr > 0, 2, arr)
# array([[-0.33064906,  2.        , -1.20929661, -0.46463464],
#        [ 2.        ,  2.        , -2.28132049, -0.07881734],
#        [ 2.        , -1.60864988,  2.        , -0.38871597],
#        [-0.49153513, -0.36618016,  2.        , -0.27166206]])
```

- Condition boolean 복합적 사용
```
cond1 = True
cond2 = False

np.where(cond1 & cond2, 0, 
         np.where(cond1, 1, 
                  np.where(cond2, 2, 3)))
```
